# Perceptron Model

------------------

## Description

- 感知机（perceptron）是二类分类的线性分类模型，即单感知机模型不能完成多分类任务；
- 感知机学习要求数据是线性可分的；
- 感知机学习旨在求出将训练数据进行线性划分的分离超平面；
- 感知机模型（perceptron model）是判别式模型 <http://www.github.com/>
